Comandos terminal de windows

    cd -> permite moverse entre carpetas
    cd .. -> permite devolverse entre carpetas 
    dir -> permite ver los archivos o carpetas que tiene un directorio
    mkdir [nombre_directorio]-> Crea un nuevo directorio.
    rmdir [directorio]-> Elimina un directorio.
    del [archivo]-> Elimina un archivo.
    copy [origen] [destino]-> Copia archivos de una ubicación a otra.
    move [origen] [destino]-> Mueve o renombra archivos o directorios.
    echo [texto]-> Imprime texto en la terminal.
    cls-> Limpia la pantalla.
    touch [nombre_archivo.extensión}-> crea un archivo con su respectiva extensión
